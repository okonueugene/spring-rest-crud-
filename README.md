# spring-rest-crud-

Boiler plate code for creating a spring rest crud application

## Features

- Spring boot
- Spring data jpa
- Spring rest
- H2 database

## Dependencies

- Spring boot
- Spring data jpa
- Spring rest
- H2 database

## How to use

1. Clone the repository

git clone https://github.com/okonueugene/spring-rest-crud.git

2. Build the project

mvn clean install

3. Run the application

mvn spring-boot:run
